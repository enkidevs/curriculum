# Writing comments in CSS

---
## Content

Single-line comments: 

```
body { 
  font-size: 1em; /*single-line comment*/
}

``` 
Multiline comments:

```
p { 
  color: red;
  /*comments may also 
   cover multiple lines */
}
```
