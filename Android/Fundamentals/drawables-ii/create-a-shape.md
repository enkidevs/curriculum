## Content

To define shape:
1.	Create a new Android XML file in the folder res/drawable.
2.	Make sure the root element of the file is <shape> as this defines a ShapeDrawable.

**Example**
```
<shape xmlns:android=
"http://schemas.android.com/apk/res/android"
    android:shape="oval">
    

```
