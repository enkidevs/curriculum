## Content

Standard way:
```
new Date().getTime()
// 1259359833574
```
Shorter way:
```
+new Date()
// 1259359833574
```
The above is using the `+` shortcut to convert the date object to a  number.

For more powerful date manipulation,
try `moment.js`.
