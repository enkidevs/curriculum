# HTML Structure Defined

---
## Content

When writing in HTML there is a certain structure you must follow.

The structure is:
(Opening tag)(Element)(Attribute)(Closing tag)

Example:
```
<a 
  href="http://enkipro.com/">Enki
</a>
```
