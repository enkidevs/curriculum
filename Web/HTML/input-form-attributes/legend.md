# Legend
---
## Content

The HTML input form `<legend>` element is used to represent a caption for the content of the `<fieldset>` element.

The `<legend>` element has to be the first child within a `<fieldset>` element.

Example:
```
<form>
  <fieldset>
    <legend>User Info:</legend>
    Name: 
      <input type="text"><br>
    Email:
      <input type="text"><br>
    Date of birth: 
      <input type="text">
  </fieldset>
  <input type="submit">
</form>
```
