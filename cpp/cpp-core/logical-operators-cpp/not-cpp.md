# NOT (!) Operator

---

## Content

The `!` operator returns the inverse of the operand's boolean value. If the operand is true, it returns false, and vice versa.

```cpp
bool a = true;

if(!a) {
  printf("a is false.")
} else {
  printf("a is true.")
}
// Output: "a is true."
```
