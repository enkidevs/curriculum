# `if/else` shortcut `conditional operator`

---
## Content

Rather than using standard `if/else` notation such as:

```
var count = 101;
var stop;
if (count > 100) {
  stop = true;
}
else {
  stop = false;
}
// stop = true
```
It is possible to use a `Condition (ternary)` operator:

```
var count = 99;
var stop = (count > 100) ? true : false;
// stop = false
```
