# Format currency in JavaScript


---

## Content

The function below will ensure numbers are correctly rounded to 2 decimal places and prevent some null errors: 

```javascript
function currency(n){
  n = parseFloat(n);
  return isNaN(n) ? false : n.toFixed(2);
}
```
