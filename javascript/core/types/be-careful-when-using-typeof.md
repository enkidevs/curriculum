# Be careful when using 

`typeof`

.


---

## Content

`typeof` returns "object" for all objects, and nothing more specific.

Examples:

```bash
typeof null // "object"
typeof new Date() // "object"
typeof new Array() // "object"
```


---

## Practice

```javascript
typeof new Date();
```
