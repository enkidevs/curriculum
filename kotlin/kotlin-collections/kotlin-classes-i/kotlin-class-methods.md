# Class Methods

---
## Content

Classes can have methods (functions) that define their behavior:

```kotlin
class Book {
    var pages: Int = 0

    fun isLongBook(): Boolean {
        return pages > 300
    }

    fun getSummary(): String {
        return "This book has $pages pages"
    }
}

fun main() {
    val myBook = Book()
    myBook.pages = 400
    println(myBook.isLongBook()) // Output: true
}
```
