# Else Statements 🌧️

---
## Content

Else statements provide an alternative action if the condition is false. 

It's like deciding to take an umbrella if it's not sunny:

```kotlin
if (temp > 30) {
    println("🕶️")
} else {
    println("☂️")
}
```

The code above prints `☂️` if `temp` is less than 30.
