# Commands to shutdown or restart the system

---
## Content

To shutdown:
```
$ sudo shutdown -h now 
$ sudo halt
$ sudo poweroff
$ sudo init 0 
```


To restart:
```
$ sudo reboot
$ sudo shutdown -r now
$ sudo init 6
```
