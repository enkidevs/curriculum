# Monitor System Memory using `vmstat`

---
## Content

`vmstat` allows the user to monitor virtual
memory statistics such as processes, memory,
paging, block IO, traps , disks and cpu
activity.

The user can specify a sampling period.

```bash
$ vmstat 1
procs

```
