## Content

To execute a command as a different user, call **sudo** with the ***-u*** argument:
```bash
$ sudo -u user command
```
Where *user* is the name of the user you want to execute the command as.

Example:
```bash
foo@linux:~$ whoami
foo

foo@linux:~$ sudo -u test whoami
test
```
