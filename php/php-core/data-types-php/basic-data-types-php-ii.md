# Basic Data Types (integer, float)

---

## Content

Integers are whole numbers, such as 1, 2, 3, 99, 5555.
```php
$number = 10;
echo $number;
// 10
```

### float

Floating-point numbers (also refered to as **double**), are numbers that contain a decimal point.
```php
$number = 10.5;
echo $number;
// 10.5
```
