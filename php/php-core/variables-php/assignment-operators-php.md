# Assignment Operators in PHP

---

## Content

In **PHP**, assignment operators are used to assign values to variables.

Other than `=`, **PHP** also includes compound assignment operators such as `+=`, `-=`, `*=`, and `/=`.

```php
// equivalent to $x = $x + 5
$x += 5;
echo $x;
// Outputs: 12

// equivalent to $y = $y * 2
$y *= 2;
echo $y;
// Outputs: 26
```
