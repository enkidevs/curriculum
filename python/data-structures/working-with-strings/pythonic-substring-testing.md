# Pythonic substring testing

Perform a substring test in a pythonic fashion using the `in` operator:
 

```python
string = 'Hello world'
if 'Hello' in string:
  print 'Substring!'
```

Instead of:


```python 
string = 'Hello world'
if string.find('Hello') != -1:
  print 'Success!'
   
```
