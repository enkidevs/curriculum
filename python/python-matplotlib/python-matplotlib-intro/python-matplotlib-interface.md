# The Matplotlib Interface

---
## Content

Let's explore how matplotlib works:

```python
import matplotlib.pyplot as plt

# Create some data
x = [1, 2, 3, 4]
y = [10, 15, 13, 17]

# Basic plotting
plt.plot(x, y)
plt.show()
```

![output](https://img.enkipro.com/7f2163278be60d21607557ad2b4ce22f.png)

> ðŸ’¡ Always use `plt.show()` to display your plot!

Matplotlib has two main ways to create plots:

```python
# Method 1: Simple state-based interface
plt.plot(x, y)
plt.title("My Plot")
plt.show()

# Method 2: Object-oriented interface
fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title("My Plot")
plt.show()
```

> ðŸŽ¨ The object-oriented interface gives you more control over your plots! It allows you to manage multiple plots and their properties independently through separate axis objects.
