# Splitting Arrays

---

## Content

Here's how to split an array into two or more arrays:

```python
# Full year test scores
scores = np.array([85, 92, 78, 88, 95, 82])

# Split into semesters
sem1, sem2 = np.split(scores, 2)
# sem1: array([85, 92, 78])
# sem2: array([88, 95, 82])

# Split into quarters
q1, q2, q3, q4 = np.array_split(scores, 4)
# q1: array([85, 92])
# q2: array([78, 88])
# q3: array([95])
# q4: array([82])
```
