# Common Methods

---
## Content

Here are some essential DataFrame methods:

```python
import pandas as pd

df = pd.DataFrame({
    'A': [1, 2, 3],
    'B': [4, 5, 6]
})

# Value counts
print(df['A'].value_counts())
# Output:
# A
# 1    1
# 2    1
# 3    1
# Name: count, dtype: int64

# Check for missing values
print(df.isna().sum())
# Output:
# A    0
# B    0
# dtype: int64

# Drop duplicates
df.drop_duplicates()
```

> 📝 These methods are some of the foundational tools for data exploration!
