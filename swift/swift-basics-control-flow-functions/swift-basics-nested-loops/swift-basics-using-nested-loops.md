# Using Nested Loops

---
## Content

A nested loop is simply a loop inside another loop.

  ```swift
for i in 1...3 {
    for j in 1...2 {
        print("i: \(i), j: \(j)")
    }
}
```

This prints each combination of `i` and `j`, showing how nested loops work.
