# Multiplication and Division

---
## Content

- **Multiplication (`*`)**: Multiplies two numbers.
  ```swift
  let product = 7 * 6 // product is 42
  ```
- **Division (`/`)**: Divides one number by another.
  ```swift
  let quotient = 20 / 5 // quotient is 4
  ```
