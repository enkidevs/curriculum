# `calc()` for simpler maths

---
## Content

For styling a 7-column grid you may use something like :
```
.column-1-7 {
   width: 14.2857%;
}
.column-2-7 {
   width: 28.5714%;
}
.column-3-7 {
   width: 42.8571%;
}
```

Use the calc() function instead to make the maths behind the layout easier to understand :
```
.column-1-7 {
   width: calc(100% / 7);
}
.column-2-7 {
   width: calc(100% / 7 * 2);
}
.column-3-7 {
   width: calc(100% / 7 * 3);
}
```

You can also mix units!
```
.mixing {
   width: calc(50% + 30px);
}

```
