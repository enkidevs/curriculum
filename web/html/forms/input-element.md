# Input Element
---
## Content

The `<input>` element is the most basic, and flexible form element. Interactive `<input>` elements like text fields, radio buttons, and checkboxes, must be placed within an `<form>` HTML element to collect user input.

Here's an example of a raw text field:
```html
<form>
 First Name:
 <input
  type="text"
  name="firstname">

 Last Name:
 <input
  type="text"
  name="lastname">
</form>
```

```html

  <input
    type="radio"
    name="age"
    value="0-18"
    checked>0-18<br>
  <input
    type="radio"
    name="age"
    value="19-25">19-25<br>
  <input
    type="radio"
    name="age"
    value="26-35">26-35<br>
  <input
    type="radio"
    name="age"
    value="36+">36+
</form>
```

Example submit:
```html
<form>
 Full name:<br>
 <input
  type="text"
  name="fullname"><br>
 <input
  type="submit"
  value="Submit">
</form>
```
