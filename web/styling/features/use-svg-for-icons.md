# Use `SVG` for icons

---
## Content

Since `SVG` (Scalable Vector Graphics) scales well for all resolution types, it is useful for icons:
```css
.logo {
  background: url("logo.svg");
}
```
`SVG` is supported in all browsers back to IE9.
