# Make text unselectable

---
## Content

This is a method in CSS to make text in a web page unselectable. 

```css
.noselect {
  user-select: none;
}
```

Once the class is assigned to a paragraph like below, the text will no longer be selectable.

```html
<p>
  Selectable text.
</p>
```
```html
<p class="noselect">
  Unselectable text.
</p>
```
