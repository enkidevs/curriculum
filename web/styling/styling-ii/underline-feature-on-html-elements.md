# Underline feature on HTML elements


---

## Content

To remove the natural underline feature that comes with `a` elements:

```css
a {
    text-decoration: none;
}
```

To underline a `span`:

```html
<span class="underline"> Span </span>
```

Use:

```css
span.underline {
  text-decoration: underline;
}
```
